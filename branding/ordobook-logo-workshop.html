ORDOBOOK Logo Directions

Six directions, all rooted in ORDO. Montserrat throughout. Each shown dark (app UI) and light (PDF/document). The goal: ORDOBOOK reads as its own product while ORDO is unmistakably its parent.

A — Weight Contrast · ORDO at 900, BOOK at 300 · No color
Dark UI · App Context
ORDOBOOK

One word, two weights. No color required — the contrast does all the work. Timeless, flexible, works at any size.

Light · PDF / Document
ORDOBOOK

Works reversed, embossed, watermarked. The most versatile direction — no color dependency.

B — ORDO in Brand Green · BOOK bold neutral · Color splits the name
Dark UI · App Context
ORDOBOOK

Green on ORDO signals the parent brand instantly. BOOK in white says "product." Clear hierarchy, strong color story.

Light · PDF / Document
ORDOBOOK

Vibrant on white. The green reads as forward momentum — fits a tool built around growth.

C — Stacked · ORDO dominant · BOOK as spaced product label
Dark UI · App Context
ORDOBOOK

ORDO takes full authority. BOOK is the product designation — small, spaced, green. Strong brand-family language.

Light · PDF / Document
ORDOBOOK

Reads clearly at small sizes — PDF footer, document watermark. Hierarchy is unambiguous.

D — Grid Icon Adapted + Wordmark · Emerald · Circles only (softer than the parent mark)
Dark UI · App Context
ORDO
BOOK

The ORDO grid icon reinterpreted — circles only, diagonal. Nods to the parent without copying it. Most distinctive at small sidebar sizes.

Light · PDF / Document
ORDO
BOOK

Icon becomes the brand anchor at small sizes. Most scalable direction.

D (alt) — Purple Accent · ORDOBOOK owns its own color within the ORDO family
Dark UI · Purple Variant
ORDO
BOOK

Purple as ORDOBOOK's dedicated color. Enables a full product palette: green = ORDO parent, purple = ORDOBOOK, cyan = next product.

●  Emerald  →  ORDO (parent brand)
●  Purple   →  ORDOBOOK
●  Cyan     →  Next ORDO product

Each product distinct. All unmistakably ORDO.

E — ORDO·BOOK with Green Midpoint Dot · Separator as sole color touchpoint
Dark UI · App Context
ORDO·BOOK

The green dot is the only color. Restrained, precise. Separates without breaking the compound name. Feels premium.

Light · PDF / Document
ORDO·BOOK

The most understated direction. Works elegantly as a document footer mark or PDF watermark.
